### 【TsumeTsume_Strategy】コアシステム仕様書

## ユニットの大まかな分類
    ユニットのクラスの最大の分類は「歩兵」と「騎乗兵」の二つ。
    「騎乗兵」はさらに「騎兵」と「飛行兵」に分けられる。
    輸送隊は騎兵に分類される（騎兵の一種。2026-07-11 変更）。
    これらの分類はすべてユニットのクラスに依拠している。

## ユニットのパラメータ
    HP、力、魔力、技、速さ、守備、魔防（魔法防御の略）の７種。
    各パラメータの詳細は各仕様の説明にて後述。

## 近接物理攻撃の三すくみ
  # 前衛武器と後衛武器の概念
    - 前衛武器の特徴
    　1. 三すくみ有り
    　2. 挟み撃ち攻撃可能
    　3. 前衛武器装備の歩兵ユニットのみは、味方の隣接ユニットが受ける挟撃を無効化する。

    - 後衛武器の特徴
    　1. 遠距離攻撃の仕様（近接攻撃不可）
    　　 - 基本2マスの範囲の敵に間接攻撃可能
    　　 - 移動せずに攻撃する場合3マス以上（武器の制限まで）遠い敵に間接攻撃が可能
    　2. 「飛翔」状態の敵ユニットに攻撃が可能。
    　3. 前衛武器装備の敵ユニットから被攻撃時、後衛武器装備ユニットは「武装無し」状態として扱われる。

  # 三すくみの仕様
    1. 前衛武器装備時のみ三すくみ有り。
    2. 三すくみ補正の計算式
        a. 三すくみ有利側が攻撃する場合
    　     ダメージ計算式 + Math.Max( 2 , Math.Max( 攻撃側の技 - 防御側の速さ ) )
        b. 三すくみ有利側が防御する場合
    　     ダメージ計算式 - Math.Max( 2 , Math.Max( 防御側の速さ - 攻撃側の技 ) )
        ※「防御側」とは、「攻撃を受ける側」の便宜的な言い換え。
    3. 後衛武器装備もしくは武装無し（非装備）で前衛武器から攻撃を受ける場合、
       三すくみ不利の判定でダメージを計算する。　
       ※後衛武器側の攻撃時は三すくみの判定無し。
  　
## 救出システム
  # 騎乗ユニットと救出システム
    1.「救出」コマンドは、ユニットと隣接時に利用可能なコマンドで、騎乗兵のみ利用可能。
       コマンド使用後はユニットに「救出」状態を付与される。
    2. 救出コマンドを利用したら隣接するユニットを1体選択。
       救出されたユニットは救出したユニットに紐付けられ格納される。
       → 各ユニットに紐づけられている救出済みユニットに格納。
         格納できるユニットは1体のみ（輸送隊クラスのユニットはその限りではない）
    4. コマンド使用後、使用していない移動力分の再移動のみ可能。
    5. 逆に騎乗ユニットは他のユニットからの「救出」「引き受け」を受けられない。
    6. 「救出」状態のユニットは「降ろす」コマンドを使用可能。
       隣接する一マスを選択して救出したユニットを配置する。
    7. 騎乗ユニットは「引き受け」コマンドを、「救出」状態のユニットと隣接時に使用可能。
       格納されている救出済みユニットを自ユニットの救出済みユニットとして格納する。
       自ユニットには「救出」状態が付与される。
       ※コマンド使用後でもまだ行動済み状態に移行しない。
         直後から「降ろす」コマンドを使用できるようにする配慮。
    8.「救出」「引き受け」「降ろす」いずれのコマンドも
    　 1回の行動可能状態時につき使用可能な回数は1回のみ。
    9. 輸送隊クラスのユニットに隣接した騎乗兵ユニットは、
    　 行動可能状態時「乗り込む」コマンドを使用可能。

  # 輸送隊クラスのユニットと救出
    輸送隊のクラスは騎兵（騎乗ユニット）に分類され、
    「救出」「引き受け」「降ろす」コマンドの仕様も継承するが、
    違いは以下の3点
    1. 救出済みユニットとして格納できる数の上限が4体。
    2. 騎乗ユニットも「救出」「引き受け」「降ろす」が可能。 
    3. 「引き受け」「降ろす」使用時は、格納された救出済みユニットたちがリスト化されたUIから
        対象となるユニットを選択して各コマンドを実行する。
    ※全てのユニットは輸送隊ユニットを「救出」することはできない。

  # 歩兵ユニットと救出システム
    1. 歩兵によるユニットの救出は不可。
       「代わりに降ろす」コマンドのみ使用可能。
    2. 「代わりに降ろす」は隣接する騎乗ユニットが救出状態の場合に使用可能で、
       救出状態のユニットと隣接したマス1つを選択して、救出されているユニットを降ろす。
    3. 「代わりに降ろす」で降ろされる対象のユニットは自ユニット以上の移動力をもつ歩兵ユニットのみ
       ※移動力が低いユニットほど多くのユニットを「代わりに降ろす」ことができるが、
　     　「代わりに降ろす」の対象にもなりにくい。
    4. 輸送隊クラスのユニットに隣接した歩兵ユニットは、行動可能状態時「乗り込む」コマンドを使用可能。

## 空中移動システム 
    1. 飛行ユニット選択時のコマンドで使用可能。
       ユニット自身に「飛翔」の状態を付与する。
       状態時は、「飛翔」状態のユニットを除いて戦闘が行われない。
       戦闘後の「飛翔」コマンドの使用は基本は不可能とする。

    2.「飛翔」状態時は敵ユニットによる移動を阻害されない。
　　　　敵ユニットもまた「飛翔」状態のユニットによる移動を阻害されない。 

　　3.「飛翔」状態時はマップのタイルの効果も受けない。

　　4.「飛翔」状態は常に2ターン持続する

　　5. 非「飛翔」状態時は騎乗ユニットと同じように扱われる。
